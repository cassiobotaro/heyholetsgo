# Bench

Comparando linguages populares

## Linguagens

* python
* JS
* Go

## Como rodar

### Node

`node app.js`
`hey http://localhost:8080`

### Python

`python app.py`
`hey http://localhost:8080`

### Go

`go run main.go`
`hey http://localhost:8080`

### PHP

`php -S localhost:8080`
`hey http://localhost:8080`

### Rust

`cargo run --release`
`hey http://localhost:3000`
